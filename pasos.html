MI PROYECTO DE TESIS

Por Cristian Valdivia

Marco Teorico

  1. 1. Se debe ingresar con la cuenta propia de github luego ir a la dirección https://github.com/cristn/prueba-git , luego hacer click en el botón llamado fork para hacer propio el repositorio en la cuenta cristn.
  2. 2. Luego se clona el repositorio ubicado en mi cuenta github “Miguel2022” ingresando el comando “git clone https://github.com/miguel2022/prueba-git.git”.
  3. 3. Se debe crear un archivo con extensión “.gitignore” EJ: archivo.gitignore y en el cual se agregue el siguiente campo: # Ignorando archivos .jpg en la carpeta images images/*.jpg
  4. 4. Se agregaron 4 paginas html: marco_teorico.html , desarrollo.html , referencias.html y pasos.html, cada uno al ser completado se agregó bajo el comando “git add (nombre del archivo)”, luego se comentó cada vez que se agregaba una página desarrollada completamente bajo el comando “git commit –m “comentario””.
  5. 5. Cree una rama con el comando “git branch diseno”, luego ingrese a la rama con el comando “git checkout diseño”, en el cual cambie el color del hover de las etiquetas a verde y lo comente, después cambie el color de a.prueba:hover a negro y lo comente, todo esto en el archivo estilos.css.
  6. 6. Para mezclar las dos ramas ingrese a la rama master “git checkout master” y aca fusione la rama master con la rama diseño con el comando “git merge diseño” y se fusionaron ambas ramas sin eliminar la rama diseño.
  7. Se agrega la actualizacion al repositorio gitbut.com/miguel2022 con el comando "git push -u origin master"
